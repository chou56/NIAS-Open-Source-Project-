# NIAS-Open-Source-Project-
「捨我其誰，成功不必在我。」 —— 本專案旨在將深奧的《唯識學》轉化為可量化的 AI 評估工具，引導人類進入「物隨心轉」的一元性平安，共同建造人間伊甸園。
📖 專案願景
NIAS (Non-dual Intelligence Assessment System) 不是一個單純的 App，而是一套心靈導航系統。我們致力於幫助用戶：

建立行為病識感：察覺損人利己行為對整體生存系統的「腫瘤化」傷害。

實踐開明自我利益：理解「利他」是更高明的「利己」。

達成境智一如：透過每日練習，從二元對立的焦慮轉向一元性的喜悅與平安。

🧩 核心模組 (Core Modules)
本專案目前規劃以下三個開源模組，歡迎社會大眾貢獻智慧：

1. 🧠 心識位階演算法 (The Ranking Algorithm)
理論基礎：依據「洞山五位」定標。

開發任務：將「正中偏」至「兼中到位」的轉化邏輯程式化。

資料夾路徑：/core/ranking-logic

2. 💬 轉識成智 AI 教練 (AI Wisdom Coach)
技術棧：Firebase Vertex AI (Gemini API)。

核心邏輯：

緩衝定標：引導用戶回到「一心位」。

病識感建立：分析衝突背後的二元損耗。

策略引導：實踐「右手給左手」的布施法門。

資料夾路徑：/functions/ai-coach

3. 📊 一元性覺察量表 (Assessment Library)
內容：包含「個人主義者職場量表」與「生活衝突量表」。

格式：JSON 格式，易於擴充與在地化。

資料夾路徑：/assets/questionnaires

🛠️ 開發者快速上手 (Firebase 部署)
複製專案：git clone https://github.com/YourOrg/NIAS-App.git

初始化 Firebase：在專案根目錄執行 firebase init。

部署 Prompt：將 /prompts/system_instruction.txt 貼入您的 AI 模型配置中。

運行測試：使用模擬器測試「職場利益衝突」的引導反饋。

🤝 參與貢獻 (How to Contribute)
我們歡迎各界人士以「布施」的心態參與，將福德轉化為改變世界的功德：

開發者：協助優化 Firebase 後端邏輯與 UI 體驗。

哲學者/心理學家：豐富量表情境，深化 AI 的點評智慧。

推廣者：協助編寫「人間伊甸園」的引導文案，開化人心。
